# Revanth Naidu Palukuri — Cinematic Portfolio

A cinematic, immersive portfolio website built with React, Tailwind CSS, Framer Motion, GSAP, Lenis smooth scrolling, and Three.js.

## Features

- **Cinematic Design**: Deep blacks, soft whites, huge typography, slow smooth animations
- **A24 Film Vibe**: Atmospheric lighting, film grain overlay, emotional pacing
- **Interactive 3D Elements**: Floating orbs, constellation skills visualization
- **Smooth Animations**: Framer Motion for cinematic transitions, Lenis for smooth scrolling
- **Responsive**: Works beautifully on desktop, tablet, and mobile

## Tech Stack

- **React** - Component-based UI
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Animation library
- **GSAP** - Advanced animations
- **Lenis** - Smooth scrolling
- **Three.js + React Three Fiber** - 3D graphics
- **Vite** - Build tool

## Sections

1. **Hero** - Cinematic introduction with 3D floating orb
2. **Objective** - Personal philosophy quote
3. **About** - Profile with stats
4. **Education** - Timeline visualization
5. **Skills** - Interactive constellation
6. **Experience** - Internship details
7. **Projects** - Featured work showcase
8. **Certifications** - Achievements
9. **Contact** - Contact form and details
10. **Footer** - Minimal closing

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
